# p2p_chat
📱 P2P Chat Application

A Peer-to-Peer (P2P) chat application that enables users to communicate directly with each other through private chats and group chats, similar to apps like IMO or Telegram.
The system focuses on real-time messaging, secure connections, and a simple user experience.

## Features
- User registration and login
- One-to-one real-time chat
- Group chat with join/leave functionality
- Group member management
- File sharing between users
- Secure peer-to-peer communication
